2. 문제점  
  - flask run을 사용하여도 오류 페이지로 연결되는 상황 발생

  - 표의 첫번째 열에 체크박스 누락  
   -> html 파일에서 각 행에 체크박스 추가하도록 코드 수정  
  - 각 제품의 성별 표기 누락  
   -> 크롤링 시 해당 데이터를 수집하지 않은것으로 확인, 실제 파일에서는 추가 할 데이터를 꼼꼼하게 확인하고 코드를 추가 확인하자.
